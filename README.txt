MergedHosts
===========
收录了Jack YouOSX的YouTube专用hosts，并整合了imouto.host、netsh.org、huhamhire-hosts、google-hosts中的所有Google服务（含YouTube）地址；
其他网址收录Wiki百科、Dropbox、Facebook、Twitter、Wordpress。

#2、为何会有218.176.242.247？本人经常使用东京节点的vpn，为了稍微减少使用其他地区ip的高ping影响，对个别地址使用日本ip，此ip国内网络无法访问。
